Proyecto AyD 2015
Users story:
“como invitado quiero dar mi nombre para jugar una partida”
“como usuario registrado quiero elegir un nivel de dificultad”
“como jugador registrado quero tener acceso a estadisticas de mis juegos y un ranking general de los usuarios registrados”
“como usuario registrado quiero sumar puntaje especial por partidas ganadas consecutivas”
“como usuario registrado quiero modos de juego, por ejemplo: el mejor de 5, el mejor de 3, simple”
“tanto como para invitado como para usuario registrado queiro jugadas con limite de tiempo”
“como usuario registrado quiero personalizar el color de fichas y fondo”
“como usuario registrado quiero ver las estadisticas con respecto a otru usuario registrado” (ver cuanteas veces jugaste con otro usuario, las derrotas, las victorias, etc)
“como usuario registrado quiero poder desafiar a usuarios de ranking similar al mio”
“como usuario registrado quero tener la posibilidad de jugar mas de una partida a la vez”
“como usuario registrado queiero tener la posibilidad de poner foto de perfil”
“como usuario registrado queiro modificar el avatar del jugador (no necesariamente el username con el que se registro)”
“como usuario registrado quiero que se pueda cambiar la contraseña”
